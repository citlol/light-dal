# Wishlist App Backend

A RESTful API backend for a wishlist sharing application.

## Features

- Express.js web server
- Health check endpoint
- Dynamic routing with URL parameters
- JSON API responses

## Installation
```bash
npm install
```

## Running the Server
```bash
node server.js
```

Server runs on: `http://localhost:3000`

## API Endpoints

- `GET /` - Welcome message
- `GET /health` - Health check
- `GET /greet/:name` - Personalized greeting
